# 🚗 加州路考英语练习 · California Road Test Phrases

一个帮助中国学员练习**加州 DMV 路考**中考官常用英语指令和口语的手机网页 app。

类似背单词的形式：每张卡片显示**英文 + 中文翻译 + 中文说明**，并可用浏览器自带语音朗读英文，支持重复播放、上一题/下一题、分类筛选、随机出题、调节语速。

## ✨ 功能

- 📱 纯静态网页，可用 **GitHub Pages** 托管，手机浏览器直接访问
- 🔊 **英语 TTS 语音朗读**（基于 Web Speech API，无需联网调用）
- 🔁 **重复播放**按钮，反复练习听力
- ➡️ **下一题 / ⬅️ 上一题**
- 🗂️ 按分类练习（开始、车辆检查、行驶中、停车与动作、路口让行、结束）
- 🙈 **隐藏中文模式**：先听英文，点「显示中文答案」再看翻译（自测听力）
- 🙉 **隐藏英文模式**：连英文原文也隐藏，纯靠耳朵听 TTS，模拟真实考试（两者可同时开启）
- 🗣️ **播放回答**按钮：需要口头回答的题目附带建议英语回答并可朗读，练习应答
- 📚 **两套题可切换**（按创建日期命名）：句式与说法不同，同一含义换着练更灵活
- 🔀 随机出题 · 🐢 语速可调（慢 / 正常 / 快）
- 📝 每套约 55 条加州路考高频指令与口语

## 🚀 如何托管到 GitHub Pages（手机访问）

**方法 A：从分支托管（最简单）**

1. 把 `index.html` 和 `questions.js` 合并到 `main` 分支。
2. 打开仓库 **Settings → Pages**。
3. **Source** 选 `Deploy from a branch`，分支选 `main`，目录选 `/ (root)`，保存。
4. 等一两分钟，页面地址形如：
   `https://<你的用户名>.github.io/practice_driving_test/`
5. 手机浏览器打开这个网址即可练习（建议加到主屏幕）。

**方法 B：用 Actions 自动部署**

本仓库已附带 `.github/workflows/deploy-pages.yml`。在 **Settings → Pages → Source** 选择 **GitHub Actions** 即可，推送到 `main` 时自动发布。

## 📱 手机使用小贴士

- iOS Safari / Android Chrome 出于安全限制，**首次需要你点一下按钮**才会发声；点「重复播放」或「下一题」即可听到。
- 想要更像真人的发音，可在手机系统里安装/启用英文（美式）语音。
- 把网页「添加到主屏幕」，用起来就像一个 App。

## ✏️ 如何添加题目

编辑 `questions.js`。题库结构为 `QUESTION_SETS`，每套用创建日期命名，往对应套的 `questions` 数组里按下面格式追加即可：

```js
{
  cat: "行驶中",                       // 分类
  en: "Turn left at the light.",       // 英文（考官说的话）
  zh: "在红绿灯处左转。",               // 中文翻译
  note: "提前打转向灯、减速观察后再转。", // 中文说明 / 什么时候会听到
  reply: "Okay.",                      // 可选：需要口头回答时的建议英语回答
}
```

想新增一整套题，按 `{ id: "日期", name: "第三套 · 日期", questions: [ ... ] }` 再加一个对象即可，界面会自动出现在题库下拉里。

> ⚠️ 题库根据网络上公开的加州路考常见指令整理，仅供英语练习参考，实际考试以考官指令和 DMV 规定为准。祝考试顺利，一把过！🎉
